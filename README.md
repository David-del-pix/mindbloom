## What is MindBloom?

MindBloom is a creative, relaxing word-garden game where every word you type blooms into a unique flower. It's designed for:

- 🧠 **Brain breaks** — refresh your mind during work or study
- 👨‍👩‍👧 **Family fun** — kids and adults can play together
- 🏆 **Friendly competition** — challenge a friend in 2-player mode
- 📚 **Vocabulary building** — longer, rarer words earn bigger flowers and more points

---

## How to Play

1. **Type any word** into the input box
2. **Press Plant or hit Enter** — your word blooms into a flower!
3. **Longer words = bigger flowers = more points**
4. **Switch to 2-Player mode** to take turns and compete
5. **Clear the garden** anytime and start fresh

### Scoring

| Word Length | Points | Flower Size |
|---|---|---|
| 2–3 letters | 1 pt | 🌱 Tiny sprout |
| 4–5 letters | 2 pts | 🌼 Small bloom |
| 6–7 letters | 3 pts | 🌺 Medium flower |
| 8–10 letters | 4 pts | 🌻 Large blossom |
| 11+ letters | 5 pts | 🌟 Giant masterpiece |

## Features

- 🌸 **Unique flowers** — every word generates a procedurally styled flower
- 🎮 **Solo & 2-Player modes** — play alone or go head-to-head
- 🔊 **Sound effects** — satisfying audio tones scale with word length
- ✨ **Particle animations** — floating emojis celebrate each bloom
- ↩️ **Undo** — remove your last flower if you change your mind
- 🔗 **Share score** — share your garden score with friends
- 📱 **Fully responsive** — works beautifully on mobile, tablet, and desktop
- ♿ **Accessible** — keyboard navigation, ARIA labels, screen reader support

---

## Tech Stack

Built with pure vanilla web technologies — no frameworks, no dependencies:

- **HTML5** — semantic structure, Canvas API for sky
- **CSS3** — custom animations, CSS variables, responsive grid
- **JavaScript (ES6+)** — SVG generation, Web Audio API, IntersectionObserver
- **Google Fonts** — Fredoka One + Nunito

## Run Locally

No build step needed — just open the file!

```bash
git clone https://github.com/David-del-pix/mindbloom.git
cd mindbloom
# Open index.html in your browser
```

Or use Live Server in VSCode for the best experience:
1. Install the **Live Server** extension in VSCode
2. Right-click `index.html` → **Open with Live Server**

## Deploy to GitHub Pages

```bash
git init
git add .
git commit -m "Initial commit: MindBloom word garden game"
git remote add origin https://github.com/David-del-pix/mindbloom.git
git branch -M main
git push -u origin main
```

Then go to **Settings → Pages → Source: main branch → Save**

Your game will be live at: `https://David-del-pix.github.io/mindbloom/`
